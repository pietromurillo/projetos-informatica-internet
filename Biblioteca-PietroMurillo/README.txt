Biblioteca Escolar - ETEC Antonio Furlan

Projeto simples em HTML, CSS e JavaScript para exibição de livros clássicos da literatura brasileira.  
Cada livro possui uma página de detalhes gerada dinamicamente via `script.js`.

---

Requisitos
- Navegador atualizado (Google Chrome, Firefox, Edge, etc.)
- Pasta `img/` contendo as capas dos livros
- Arquivos principais:
  - `index.html`
  - `livro.html`
  - `style.css`
  - `livrostyle.css`
  - `script.js`
